# OKC-Stake-Girls-Camp-2026

This repository contains a Progressive Web App for OKC Girls Camp 2026 titled Walk With Me.

## What is included
- Installable mobile-first PWA experience
- Offline-ready home screen and station flow
- Local progress saving for completed stations
- Offline asset testing page
- Service worker and manifest for installability
- GitHub Pages deployment workflow

## Project structure
- docs/ — app shell, styles, JavaScript, manifest, service worker, and offline assets
- .github/workflows/deploy-pages.yml — GitHub Actions workflow for Pages deployment

## Deploy to GitHub Pages
1. Push this repository to GitHub.
2. In the repository settings, enable GitHub Pages.
3. Choose GitHub Actions as the deployment source.
4. The workflow in .github/workflows/deploy-pages.yml will publish the contents of docs/.

## Local preview
Open the docs/ directory in a browser or serve it with a simple static server.
